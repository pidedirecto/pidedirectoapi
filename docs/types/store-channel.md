### StoreChannel {#store-channel}

Store channel that a menu should be visible for.

| Value | Description |
| ----- | ----------- |
| `PIDEDIRECTO` | PideDirecto ecommerce platform of the store |
| `PIDEDIRECTOPOS` | Point of sale in the restaurant |
| `PIDEDIRECTOKIOSK` | Kiosk at the restaurant |
| `UBER_EATS` | Uber Eats platform |
| `DIDI_FOOD` | Didi Food platform |
| `RAPPI` | Rappi platform |
| `PEDIDOS_YA` | Pedidos Ya platform |
