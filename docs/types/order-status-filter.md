### OrderStatusFilter {#order-status-filter}

Optional filter for listing orders by status.

| Value | Description |
| ----- | ----------- |
| `NEW` | New orders |
| `CANCELLED` | Cancelled orders |
| `REJECTED` | Rejected orders |
| `ACCEPTED` | Accepted orders |
| `COMPLETE` | Completed orders |
| `PENDING` | Pending orders |
| `RETURNING` | Returning orders |
| `RETURNED` | Returned orders |
| `RETURNED_BY_CUSTOMER` | Returned by customer |
| `MERGED` | Merged orders |
| `CLOSED` | Closed orders |
